## 📌 Project Overview
The goal of this project is to implement fundamental DSA concepts from scratch while simultaneously solving competitive problems on LeetCode and GFG.

## 📂 Repository Structure
To keep code organised and maintainable, I divided it into folders:

- ### 🛠️ **Automated Solutions (LeetCode/GFG)**
  * Problems synced directly via **LeetHub** and **GfG-to-GitHub** extensions.
  * Categorized by Difficulty (Easy, Medium, Hard).

- ### 🔍 **searching/**
  * **Linear & Binary Search:** Complexity $O(n)$ and $O(\log n)$.

- ### 📊 **sorting/**
  * **Classic Sorts:** Bubble, Selection, Insertion ($O(n^2)$).
  * **Advanced Sorts:** Merge Sort ($O(n \log n)$) using Divide & Conquer.

- ### 🔄 **recursion/**
  * Fibonacci, GCD, Nth Stairs, and **Recursive Linked List Ops**.

- ### 🔗 **linked_list/**
  * **Singly, Doubly, Circular, & Doubly Circular.**
  * **Advanced:** Loop Detection (Floyd's & Hashing), Palindrome Check, Merging, and Sorting.

- ### 📚 **stacks/ & queues/**
  * Complete Menu-Driven implementations including **Circular Queues**.
